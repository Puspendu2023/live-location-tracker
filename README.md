# 📍 LiveTrack - Real-Time Location Tracking System

A full-stack GPS location tracking application with real-time updates, interactive maps, and comprehensive analytics.

![LiveTrack](https://img.shields.io/badge/Status-Active-success)
![License](https://img.shields.io/badge/License-MIT-blue)

## 🚀 Features

- ✅ Real-time GPS tracking (2-5 second updates)
- ✅ Interactive map with OpenStreetMap
- ✅ Socket.IO live updates
- ✅ User authentication (JWT)
- ✅ Location history & analytics
- ✅ Geofencing (circular & polygon)
- ✅ Admin dashboard
- ✅ Dark mode support
- ✅ Responsive design
- ✅ PWA support

## 🛠️ Tech Stack

### Frontend
- React 18 + Vite
- Tailwind CSS
- Framer Motion
- React Leaflet
- Socket.IO Client
- Zustand

### Backend
- Node.js + Express
- Socket.IO
- PostgreSQL + Prisma
- JWT Authentication
- Bcrypt
